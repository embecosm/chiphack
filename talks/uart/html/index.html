UART¶

Two wire serial connection protocol

  • Rx wire to receive

The EHW5 UART: Set to 3.3V!¶

UART protocol¶

The UART protocol sends a start bit, 8 data bits and a stop bit

The idle state is high, so start bit is low. The stop bit is high.
